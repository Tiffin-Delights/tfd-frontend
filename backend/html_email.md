# Email layout structure (Tiffin Delight version)

## 1️⃣ Header block (branding strip)

Top section:

```
Centered Logo
Background: very light green gradient
Spacing: generous padding
```

Visual feel:

```
[ TIFFIN DELIGHT LOGO ]
Subscription Tiffin Service
```

Purpose:

Instant recognition → prevents phishing suspicion.

---

# 2️⃣ Title section

Main heading:

```
Reset your password
```

Subheading tone (brand voice):

```
Let’s get you back to your meals quickly.
```

Font suggestion:

```
Heading color → deep green
Weight → bold
Size → 24–26px
Alignment → center
```

Feels warm, not corporate.

---

# 3️⃣ Explanation paragraph

Short message:

```
We received a request to reset your password for your Tiffin Delight account.
Use the verification code below to continue.
```

Tone rule:

Friendly kitchen service, not banking security department.

---

# 4️⃣ OTP highlight block (MOST IMPORTANT)

Center card style:

```
Background: light green tint
Border radius: 8px
Padding: large
Shadow: subtle
```

Example appearance:

```
Your verification code

  482913
```

Typography spec:

```
Font size → 34px
Letter spacing → wide
Color → dark green
Weight → bold
```

Users should spot code instantly even while half-asleep before lunch break.

---

# 5️⃣ Expiry notice

Below OTP:

```
Valid for 5 minutes only.
```

Color:

```
Muted gray-green
```

Keeps urgency without panic.

Food brands rarely say this but modern apps must.
